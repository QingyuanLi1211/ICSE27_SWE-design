# Benchmark Construction Pipeline

This directory contains the public construction pipeline materials.

- `prompts/` contains the four task prompts used by the LLM-based construction pipeline.
- `src/` contains the deterministic outer orchestration code.
- `run_pipeline.py` is the thin entry point for the construction pipeline.
- `docs/` and `examples/` contain auxiliary pipeline contracts and an example input instance when available.

The example inputs in this package are intended to document the construction interface and artifact format.
